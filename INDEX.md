# Demo index

Every demo shipped in this cookbook, in chronological order. Hand
maintained for now. Automation lands in M2 (a PostToolUse hook will
update this file when a demo is added).

When a new demo ships, append a row at the **bottom** of the table below
and link to its folder under `demos/`.

| Date | Title | Slug | Category | Description | Status | Link |
| --- | --- | --- | --- | --- | --- | --- |
| _none yet_ | _Demo #1 lands during M0.5_ | | | | | |

## Categories

Demos are tagged with one of these categories. Add new categories only
when no existing one fits.

- Receptionist & Booking
- Drive-thru & Ordering
- Customer Support
- Companion & Coaching
- Education
- Data Extraction
- Tool Calling
- Multilingual
- Multi-Agent
- Telephony & SIP

## Status values

- `shipped`: published, has a working README, blog, reel, and run instructions.
- `in-progress`: branch open, not yet merged.
- `archived`: kept for reference but no longer maintained.
